📋 Project Overview

This repository contains the complete coursework for the Cognifyz Technologies Front-End Development Internship Program. The project demonstrates progressive front-end skills from basic HTML/CSS to advanced API integration, responsive design, and CSS preprocessing.

Technologies Used:

    HTML5, CSS3, JavaScript (ES6+)

    Bootstrap (Task 7)

    Sass/SCSS (Task 8)

    Public APIs (JSONPlaceholder)

    Responsive Design (Media Queries)

🎯 Internship Tasks Completed
Level 1: Beginner

    Task 1: Basic HTML structure with proper semantic elements, headings, paragraphs, and images

    Task 2: Inline CSS styling for fonts, colors, and backgrounds

Level 2: Intermediate

    Task 3: Responsive design using CSS media queries and separate styles.css

    Task 4: Interactive JavaScript button that changes background color on click

Level 3: Advanced

    Task 5: API integration using fetch() with JSONPlaceholder, dynamic DOM updates

    Task 6: Styled forms with client-side validation and user feedback

Level 4: Expert

    Task 7: Bootstrap framework integration for component-based styling

    Task 8: Sass preprocessing with variables, nesting, and compiled CSS
